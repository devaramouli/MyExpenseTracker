# 💰 Dynamic Expense Tracker

A lightweight, object-oriented Command-Line Interface (CLI) application built in Python to track, categorize, and audit individual expenses dynamically over time.

## 🚀 Key Features
* **Object-Oriented Architecture:** Models individual tracking records using a decoupled `person` class.
* **Dynamic Range Inputs:** Allows the user to dictate exactly how many transactions they wish to log per session.
* **Complex Data Parsing:** Implements native Python `datetime` formatting (`strptime`) to accurately convert string-based dates into queryable objects.
* **Multi-Tiered Aggregation Engine:** Extracts specific `.year`, `.month`, and `.day` structures to run time-series expense audits.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Core Modules:** `datetime`

## 📖 How It Works
1. Create a `person` session profile by submitting a target identifier name.
2. Initialize the dynamic data array engine by determining total item tracking cycles.
3. Query the aggregate calculation functions sequentially to fetch real-time financial tracking updates.
